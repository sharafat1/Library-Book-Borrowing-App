# Library-Book-Borrowing-App

This Library Book Borrowing Application was built as a part of the CAB303 studies requirements. This application allows users to check available movies in a database and then borrow movies based on available movies. Allows shows people popular movies etc. 

# How to Run Guide

Step 1: To run this application, go the App directory, open the project in visual studio 2019.

Step 2: Once the project has opened, then run the project. 

Step 3: You will see another windows pops up. 

Step 4: You will be able to follow the instructions on how to use the application in the app. If you don't know how to use it. Then check out the how to guide in the report which is inside the Library-Book-Borrowing-App directory.
